# C para Iniciantes

*Um guia progressivo e prático para aprender programação em C do zero.*

## 🚀 Introdução

Este repositório foi criado para ajudar você a dar os primeiros passos na programação utilizando a linguagem **C**, mesmo que você nunca tenha programado antes.

Aqui você encontrará um caminho estruturado, com aulas progressivas que combinam **explicações teóricas**, **exemplos comentados** e **atividades práticas**, permitindo que estude de forma autônoma e no seu ritmo.

## 🧭 Como estudar usando este repositório

**1️⃣ Faça um fork do repositório**

Clique no botão **Fork** no canto superior da página para criar sua cópia pessoal.

**2️⃣ Estude o conteúdo das aulas**

Leia com atenção o arquivo `README.md` de cada aula e execute os exemplos de código disponibilizados.

Siga a ordem das aulas, pois os conteúdos são **cumulativos**.

**3️⃣ Resolva as atividades práticas propostas**

As atividades práticas são disponibilizadas como **Issues** no repositório. 

A entrega deve ser feita utilizando **Pull Requests** no seu fork.

## 🛠️ Ambiente de desenvolvimento

Para acompanhar o curso, você precisará de:
| Ferramenta | O que é? | Recomendação |
| :--- | :--- | :--- |
| Editor de código | Onde você escreverá seu código. | [Visual Studio Code](https://code.visualstudio.com/) | 
| Compilador | Transforma o código em programa. | **Windows:** [w64devkit](https://github.com/skeeto/w64devkit/releases) <br> **Linux/macOS:** gcc/clang (já incluso) |
| Versionador | Salva o histórico de alterações do código. | [Git](https://git-scm.com/) |
